
#### 类地址栏搜索说明

搜索引擎在`searchbar.js`中定义，下面是支持的搜索引擎：

```
bd 百度
db 豆瓣
zh 知乎
wx 微信
tb 淘宝
gh GitHub
gg Google
ddg DuckDuckGo
```
